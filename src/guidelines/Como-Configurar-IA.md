# 🤖 Como Configurar a IA do DigiApp

## 📍 Onde Encontrar as Configurações

### Opção 1: Pelo Chat (Mais Fácil!)
1. Role até o final da tela onde está o **companheiro digital**
2. Localize a caixa de chat com o texto "💬 Chat com IA"
3. Clique no botão **⚙️ Config** no canto superior direito do chat
4. O modal de configurações abrirá automaticamente

### Opção 2: Verificando se a IA está Ativa
- Se você NÃO vê o botão "⚙️ Config", a IA pode estar desativada
- Para ativar: clique no botão 💾 **File** (canto superior) → ative o toggle "Use AI Chat"

---

## ⚙️ Configurações Disponíveis

### 🎭 **Tom de Voz**
Define como seu companheiro digital fala com você:

- **😊 Casual**: Linguagem descontraída brasileira
  - Exemplo: _"E aí! Bora completar tarefas hoje?"_
  
- **⚡ Energético**: Muito animado e empolgado!
  - Exemplo: _"BOOORA! VAMOS DESTRUIR ESSAS TAREFAS! 🔥🔥"_
  
- **🧘 Calmo**: Tranquilo, sereno e sábio
  - Exemplo: _"Vamos com calma... um passo de cada vez."_
  
- **🎮 Brincalhão**: Divertido e cheio de piadas
  - Exemplo: _"Ei, essa tarefa é mais fácil que derrotar um Agumon! 😄"_

---

### 😊 **Intensidade de Emojis**
Controla quantos emojis seu Digimon usa:

- **🚫 Nenhum**: Sem emojis (apenas texto)
- **😊 Baixo**: 1 emoji por mensagem
- **😊✨ Médio**: 2-3 emojis por mensagem (padrão)
- **😊✨🔥 Alto**: 4-6 emojis (bem expressivo!)

---

### ⚡ **Tamanho da Resposta**
Define o comprimento das mensagens:

- **📱 Curto**: Até 80 caracteres (mensagens bem diretas)
- **💬 Médio**: Até 150 caracteres (mais contexto)

💡 **Dica**: Respostas curtas são melhores para mobile!

---

### 💪 **Estilo de Motivação**
Como seu companheiro te motiva:

- **🌟 Encorajador**: Sempre positivo
  - _"Você consegue! Cada passo importa! ✨"_
  
- **🔥 Desafiador**: Te provoca de forma amigável
  - _"Só isso? Aposto que você consegue fazer MAIS! 💪"_
  
- **💚 Acolhedor**: Muito carinhoso e empático
  - _"Tá tudo bem. Estou aqui pra te apoiar sempre! 💚"_
  
- **⚖️ Equilibrado**: Mix de tudo (padrão)
  - Adapta o estilo baseado na situação

---

### 🏷️ **Palavras-chave Personalizadas**
Campo de texto livre para instruções customizadas!

**Exemplos do que você pode colocar:**

```
Sempre me chame de "parceiro"
Use gírias cariocas
Fale sobre programação quando possível
Seja mais formal comigo
Evite usar a palavra "tarefa", use "missão"
Mencione Digimon Adventure quando relevante
Use humor sarcástico ocasionalmente
```

💡 **Dica**: Quanto mais específico, melhor!

---

### 🌡️ **Criatividade (Temperature)**
Controla o quão criativa e imprevisível será a IA:

- **0.5-0.6**: Conservador (respostas consistentes e previsíveis)
- **0.7-0.8**: Balanceado
- **0.85**: Padrão (criativo mas controlado)
- **0.9-1.0**: Muito criativo (respostas variadas e surpreendentes)

⚠️ **Atenção**: Valores muito altos podem gerar respostas estranhas!

---

## 💾 Salvando e Aplicando

1. Após ajustar suas preferências, clique em **✅ Salvar**
2. As configurações são salvas automaticamente no navegador
3. Teste conversando com seu Digimon no chat!
4. Se não gostar, volte e ajuste ou clique em **🔄 Resetar** para padrão

## 🔧 Troubleshooting (Resolução de Problemas)

### ❓ "Não vejo o botão ⚙️ Config"
**Solução**: 
1. Vá em File (💾) no menu superior
2. Ative "Use AI Chat"
3. O botão de configuração deve aparecer

### ❓ "A IA não está seguindo minhas instruções"
**Soluções**:
1. Seja mais específico nas "Palavras-chave Personalizadas"
2. Aumente a criatividade se quiser mais variação
3. Diminua a criatividade se quiser mais consistência
4. Teste várias mensagens - a IA aprende com o contexto

### ❓ "As respostas estão muito longas"
**Solução**: 
- Mude "Tamanho da Resposta" para "Curto"

### ❓ "As respostas estão muito repetitivas"
**Solução**: 
- Aumente o slider de "Criatividade" para 0.9

### ❓ "Quero voltar ao padrão"
**Solução**: 
- Clique em **🔄 Resetar** no modal de configurações

## 📊 Como as Configurações Afetam a IA

As suas configurações são enviadas para o servidor junto com cada mensagem e influenciam:

1. **O System Prompt**: Instruções base de como o Digimon deve se comportar
2. **Temperature**: Criatividade da geração de texto
3. **Max Tokens**: Limite de caracteres da resposta
4. **Personality Traits**: Traços de personalidade baseados no ramo evolutivo

### Prioridade das Configurações:
```
1. Palavras-chave Personalizadas (maior prioridade)
2. Ramo Evolutivo (Virus/Data/Vaccine)
3. Humor do Digimon (Happy/Tired/Idle)
4. Tom de Voz escolhido
5. Estilo de Motivação
6. Intensidade de Emojis
```

---

## 🎨 Dicas de Ouro

✅ **Experimente!** Não tenha medo de testar diferentes combinações
✅ **Seja específico** nas palavras-chave personalizadas
✅ **Ajuste gradualmente** - mude uma coisa de cada vez
✅ **Respeite o ramo evolutivo** - a personalidade base vem dele
✅ **Lembre-se do humor** - Digimon cansado age diferente de feliz

❌ **Não exagere** na criatividade (acima de 0.95)
❌ **Não coloque instruções contraditórias** nas palavras-chave
❌ **Não espere perfeição** - é uma IA, pode ter variações

## 🚀 Para Usuários Avançados

### Entendendo os Bastidores

As configurações modificam o **system prompt** que é enviado para a API do Groq (LLaMA 3.1):

```javascript
// Exemplo de como suas configurações são aplicadas:
{
  tone: 'energetic' → "Seja MUITO animado e energético!"
  emojiIntensity: 'high' → "Use vários emojis (4-6)!"
  motivationStyle: 'challenging' → "Desafie o usuário!"
  customKeywords: "sempre me chame de campeão"
  temperature: 0.9 → Geração mais criativa
  max_tokens: 80 → Respostas curtas
}
```

## 📝 Notas Técnicas

- As configurações são salvas em `localStorage` como `digiapp-ai-settings`
- Cada mensagem envia as configurações para o servidor
- Não há histórico de conversa (stateless) para economizar tokens
- A API usada é **Groq** com modelo **LLaMA 3.1 8B Instant** (gratuito)
- Fallback automático para respostas programadas se a API falhar
