# DigiApp 🧬🎮

DigiApp is a **gamified productivity Progressive Web App (PWA)** inspired by virtual pets, monster evolution, and retro digital companions.

Real-life actions generate progress inside the app — tasks, routines, and consistency directly affect the evolution and state of your digital creature.

---

## ✨ Core Concept

- Complete real-world tasks
- Earn progress automatically
- Your digital companion evolves over time
- No grind, no punishment — consistency matters more than volume

---

## 🧠 Design Philosophy

- **Automation-first:**  
  On mobile, the app saves and restores progress automatically.
- **Offline-friendly:**  
  Core features work even without an internet connection.
- **Minimal friction:**  
  No forced file handling on Android.
- **Cross-platform:**  
  Desktop and mobile supported via PWA.

---

## 📱 Platforms

- Android (via Google Play – Trusted Web Activity)
- Mobile PWA
- Desktop (Chrome / Edge / Firefox)

---

## 🛠 Tech Stack

- Frontend: Vite + modern JS
- PWA: Web App Manifest + Service Worker
- Hosting: Cloudflare Pages
- Android packaging: PWA Builder (TWA)
